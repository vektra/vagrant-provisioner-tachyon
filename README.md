# Vagrant::Provisioner::Tachyon

A provisior plugin for tachyon

## Installation

Add this line to your application's Gemfile:

    gem 'vagrant-provisioner-tachyon'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install vagrant-provisioner-tachyon
